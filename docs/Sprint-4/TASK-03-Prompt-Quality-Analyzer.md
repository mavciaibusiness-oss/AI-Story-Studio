# Vision

Prompt Quality Analyzer transforms AI Story Studio from a prompt generator into a prompt evaluator.

Instead of only creating prompts,

AI understands whether the generated prompt is actually capable of producing a high-quality image or video.

The objective is not quantity.

The objective is prompt quality.

---

# Goal

Automatically analyze every generated prompt before production.

Detect weak prompts.

Suggest improvements.

Repair prompts automatically.

Generate production-ready prompts.

---

# Problem

Current AI image generators accept almost every prompt.

However:

A prompt may be

too short

too generic

missing lighting

missing camera angle

missing emotion

missing composition

missing consistency

The result becomes random.

Creators usually discover this only after spending credits.

---

# Solution

Every prompt generated by AI Story Studio will pass through the Prompt Quality Analyzer before being used.

Workflow

Generate Prompt

↓

Analyze Prompt

↓

Calculate Quality Score

↓

Suggest Improvements

↓

(Optional)

Auto Rewrite

↓

Production Ready
